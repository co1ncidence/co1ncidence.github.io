Some of My Favorite Software

08-17-2020 #Opinion #Software #Showcase

Just a list of some software I find nice to use, most are pretty minimal as well:
cfm Simple and fast terminal file manager
maim Extremely powerful CLI screenshot tool
SXIV “Simple X Image Viewer”
giph Customizable and easy to use GIF recorder
birch Freenode IRC client written in bash
farge Finally a lightweight CLI colorpicker
ungoogled-chromium Browse safely and privately, no compromises
ST Simple Terminal
shfm Terminal file browser written in bash
epy Terminal based Ebook reader
wpm Command line typing test
cchat-gtk GTK+ frontend for cchat’s
