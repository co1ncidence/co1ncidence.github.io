Some Cool Software

  • Aug 17, 2020
  • #Opinion , #Software , #Showcase
  • One minute read

This is a list of some of the best software that I have come across in my journey using Linux. I highly recommend that you try them out and maybe even give the developers a well earned star on Github to show your support :) I’ve divided the software into sections based on their uses, so if you don’t see what you need just scroll down until you see the type of software you are searching for.

Links to the software’s Github page are in the headers

Maim, the better screenshot tool

One of the most important pieces of software you can have is something to capture you display, while scrot does the job quite well, it has issues with selecting areas and windows, which is where Maim comes in, it does basically the same thing as scrot but without any of the rendering and selection issues. Fun fact, every screenshot you see on this blog was taken using Maim.

SXIV, the image viewer that, well, views images

Giph, easily record gifs of your screen

Scr, a small script to easily take screenshots and record videos

Ripcord, an dangerously lightweight and minimal Discord/Slack client

Polybar, an easily configured, infinitely customizable desktop statusbar

Lyrebird, Simple and easy to use Voice Changer for Linux
