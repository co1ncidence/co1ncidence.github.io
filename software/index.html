Some of My Favorite Software

Published: 08/17 â€” 2020 270 Words, 2 Min.

Just a list of some software I find nice to use. A lot of these correlate with my stars on github, so be sure to check them out as well if you want some other nice things to use. My preferences with software usually boil down to:

  • It gets the job done
  • It’s smaller in size than similar software
  • It’s free and open source
  • It’s not overrated normie shit
  • And it integrates well with my workflow

Now with that out of the way, here is the list:

Software Description
dash Fast and powerful POSIX shell
giph Customizable and easy to use GIF recorder
maim Extremely powerful CLI screenshot tool
mpv Simple yet featureful CLI media player
SXIV “Simple X Image Viewer”
birch Freenode IRC client written in bash
herbe Simple, yet DBus free notifications
farge Simple Colorpicker written in bash
ung-chromium Browse safely and Google-free, no compromises
imgp Blazing fast CLI image resizing and conversion
ST Simple Terminal
scr Super Crappy SCReenshot SCRipt for Sound Cloud Rappers
vimtex Make LaTeX in Vim a breeze to edit
ale A complete syntax checker and linter for Vim
neovim Objectively the best editor
shfm Terminal file browser written in posix shell
KISS Linux An independent Linux Distro focused on simplicity
fff Fucking Fast File manager written in bash
epy Terminal based Ebook reader
PDF Viewer Plus The best mobile PDF viewer there is
ripcord Not great, but it’s a minimal discord client
sls Extremely Simple su utility
gottagofast Simple and minimal CLI typing test
monkey-type Clean, fun, and minimal typing test
wmutils Very extensible set of WM utilities
0x0 Effective, No-Bullshit file hosting
